Druk alle tekens uit een tekst af die ingesloten zitten tussen vierkante haken.

### Invoer

Een regel tekst.

### Uitvoer

Alle tekens van de tekst die tussen vierkante haken staan.

### Voorbeeld

#### Invoer:

```
And no[w] for s[o]methi[n]g completely [d]iff[er]ent.
```

#### Uitvoer:

```
wonder
```
