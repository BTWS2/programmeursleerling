Als er iets fout zit in code, geeft Python meestal een foutmelding. Dit zijn vaak syntaxfouten, die aangeven dat er iets fout zit in de vorm van je code (bijvoorbeeld, voor de code uit de vorige oefening kreeg je een SyntaxError).

Er zijn ook runtime errors, die aangeven dat je code op zich syntactisch correct lijkt, maar dat er iets fout is gegaan bij de uitvoering ervan. Een goed voorbeeld is de ZeroDivisionError, die aangeeft dat je probeerde te delen door nul (wat niet mag, zoals je weet). Schrijf een programma dat een ZeroDivisionError genereert als je het uitvoert.
